# Jobstartbörse

Management system for the Jobstartbörse job fair. It manages the administration of exhibitors and job fairs while providing a self-service portal for exhibitors.

## Stack

Built with [Laravel 12](https://laravel.com) and [Filament 4](https://filamentphp.com).

## Setup

```bash
composer setup
npm install
npm run build
```

If you want to seed some default data run `php artisan db:seed`

### Development

```bash
composer dev
```

This starts the server, queue worker, logs, and Vite dev server concurrently.
